#ARRAYS
- In JavaScript, arrays are objects, not pure lists.
- JS does NOT throw error on invalid index , This causes silent bugs
- Mutating methods change the original array


## mutating-
1. A mutating operation modifies the original object/value.
2. The data is changed in place
3. After the operation, the original variable is different

## Non-mutating-
1. A non-mutating operation does NOT change the original data.
2. Instead, it returns a new value.
3. Original data stays the same
